# Continual Learning — POC umbrella

Each POC is its own git repo, tracked here as a submodule. Clone with `--recurse-submodules`.

- `poc/live_stream_stability/` — VLM continual-learning on a 35-day IRL livestream.
- `poc/recursive_finetuning_stability/` — recursive self-SFT: a coding researcher fine-tuned on its own grounded autoresearch execution traces, looped V0→V_N (weights vs. context).
